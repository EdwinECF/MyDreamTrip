# Japan Dream Trip

A travel-inspired landing page built from scratch with HTML and CSS as a front-end practice project. The website showcases some of the places I would like to visit in Japan while helping me improve my CSS layout and styling skills.

## Overview

The goal of this project was to strengthen my understanding of modern CSS by creating a clean and visually appealing webpage without relying on frameworks or templates. It focuses on responsive layouts, typography, spacing, and Flexbox positioning.

## Features

- Responsive full-width destination images
- Custom typography using Google Fonts and local fonts
- Flexbox-based layouts for content alignment
- Styled call-to-action buttons
- Fixed title element
- External links to learn more about each location
- Clean and organized HTML structure
- Modern CSS styling from scratch

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Custom Fonts

## Destinations Included

- Tokyo
- Osaka
- Hiroshima
- Mount Fuji

## Purpose

This repository is part of my web development learning journey and portfolio. It was created to practice CSS fundamentals and improve my ability to build attractive, well-structured web pages from scratch.
